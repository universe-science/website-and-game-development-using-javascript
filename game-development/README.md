# 🕹️ Разработка веб-игр и сайтов на JavaScript

Добро пожаловать в репозиторий курса **«Website and Game Development using JavaScript»**!  
Здесь ты найдёшь все необходимые инструкции, чтобы подготовить свой компьютер к занятиям, а также материалы и примеры кода, которые мы будем использовать на уроках.

Этот курс создан специально для детей и школьников, которые хотят научиться создавать свои собственные веб-сайты и игры с помощью современного языка программирования JavaScript.



## 🎯 Для кого этот курс

- Для новичков, которые никогда не программировали.
- Для тех, кто хочет создавать игры прямо в браузере.
- Для всех, кто хочет научиться работать с кодом, Git и современными инструментами.



## 📚 Чему мы научимся на курсе

- **Основы JavaScript** – переменные, функции, циклы, условия.
- **Создание веб-страниц** – HTML, CSS, стилизация.
- **Взаимодействие с пользователем** – обработка кликов, формы, события.
- **Разработка игр** – анимация, управление персонажем, коллизии, физика.
- **Работа с графикой** – рисование на Canvas.
- **Создание серверов на Deno** – обработка запросов, маршрутизация.
- **Публикация проектов** – деплой на GitHub Pages, использование туннелей.

Всё это мы будем изучать на практике, создавая реальные проекты: от простого сайта-визитки до полноценной браузерной игры.
