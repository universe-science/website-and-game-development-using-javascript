# 🕹️ Разработка однопользовательских 2D игр на JavaScript

Добро пожаловать в репозиторий курса **«2D Single-Player Game Development using JavaScript»**!  

Этот курс создан специально для детей и школьников, которые хотят научиться создавать свои собственные однопользовательские (для одного игрока) 2D игры с помощью современного языка программирования JavaScript.



## 🎯 Для кого этот курс

- Для новичков, которые никогда не программировали.
- Для тех, кто хочет создавать игры прямо в браузере.
- Для всех, кто хочет научиться работать с кодом, Git и современными инструментами.



## 📚 Чему мы научимся на курсе

- **Основы JavaScript** – переменные, функции, циклы, условия.
- **Основы веб-страниц** – базовые знания HTML, CSS.
- **Взаимодействие с пользователем** – обработка событий мыши и клавиатуры.
- **Разработка игр** – анимация, управление персонажем, коллизии, физика.
- **Работа с 2D графикой** – рисование на Canvas.
- **Работа со звуком** – воспроизведение звуковых файлов и управление звуком.
- **Публикация проектов** – деплой на GitHub Pages, использование туннелей.

Всё это мы будем изучать на практике, создавая реальные проекты: от простых игр до вашей собственной однопользовательской полноценной браузерной 2D игры.
